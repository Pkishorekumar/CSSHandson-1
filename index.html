What is CSS?

  1. CSS stands for Cascading Style Sheet.
  2. CSS describes how HTML elements are to be displayed on screen,paper,or in other media
  3. Css is not neccesary Css savs a lot of work.It can control the layout of multiple web pages alll at once
  4. External stylesheets are stored in CSS files

CSS can be added to HTML documents in 3 ways:

  • inline - by using the style attribute inside HTML elements 1
  • internal - by using a style element in the head section
  • External - by using a link element to link to an external CSS file.
  • The most common way to add CSS, s to keep the styles in external CSS files.However, in this tutorial we will use inline and internal styles,because this is easier to demonstrat, and easier for you to try it yourself.

When comparing CSS class vs ID,the difference id that CSS class applies a style to multiple elements.ID,on the other hand,applies a style to one unique element.ID is also special in thet you can use a special URL to link directly to an element and it's used by JavaScript.
In CSS, selection are used to target a special element or range of element on a web page.Once an element has been targeted,a style or set of style can be applied to the element
There is a wide range of selectors available.Two of the most commonly used are class and ID.Both are usedd to target element to which a style should be applied.
